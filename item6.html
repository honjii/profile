友達マッチングアプリFILL(チーム制作)

作品URL: https://github.com/honjii/HACKU2020-1

使用技術: html CSS JavaScript PHP mysql

担当:企画 デザイン フロントエンド

作品について

2年生の時に学外コンテストで開発しました。 なぜ友達マッチングアプリ開発しようと思ったかというと、ちょうどコロナウイルスが流行り始めた時期で、今後友達と会う機会が減ったり、疎遠になったりすることが予想されたので、そんな時に友人を誘いやすくなるようなアプローチができるアプリがあれば便利だと思ったからです。 企画では、自分の登録した友達間で暇な日をカレンダーで共有しておけば、遊びに誘いやすくなったり、会う機会が増やせるのでは？と考えました。 デザインでは、UI的にわかりやすくするためにフレンドと暇な日が一致した場合は、カレンダーの日付に色をつけることによって簡単にマッチングした日がわかるようにしました。 フロントエンドはhtml css JavaScriptでのカレンダーの動作などを担当しました。 ※当作品のリポジトリでは、他言語での再開発時を考慮し、バックエンドの機能を除いています。
