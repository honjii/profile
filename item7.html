ミスタードーナツ1000円ガチャ(個人制作)

作品URL: https://github.com/honjii/misudo_gatya

使用技術: html CSS Javascript Node.js alpine.js axios

担当: 全て

作品について

ミスタードーナツの公式サイトをスクレイピングし、商品の情報や画像などのデータを取得しました。サイトの商品の合計の値段が1000円以下になるような1000円ガチャ機能も実装しました。フロントエンドはalpine.jsを使用し、バックエンドはnode.jsで作り、フロントとバックの連結はaxiosを使用しました。
